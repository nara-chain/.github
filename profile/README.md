# Nara Solana — A Privacy-Native, Solana-Compatible Layer-1

**Nara Solana** is a next-generation Layer-1 blockchain, fully compatible with the Solana programming model while extending it with built-in privacy, larger account capacities, lower fees, and enhanced developer incentives.  
It is designed for developers who want Solana’s performance **without Solana’s limitations**, and for users who need **first-class privacy** across identity, assets, and applications.

# 🔐 Privacy as a First-Class Primitive

Nara Solana includes a **native privacy layer** with zero-knowledge proofs tightly integrated into the account and PDA model.

---

## 📛 zkID — Anonymous Identity Layer

A user-controlled, privacy-preserving on-chain identity.

- Create a zkID (e.g., `abc`)  
- Unlinkable from wallet addresses  
- Identity ownership proven via ZK proofs  
- Suitable for governance, voting, reputation, and more  

---

## 💸 Anonymous Transfers — zkID → zkID

Send assets **without revealing sender or recipient addresses**.

Features:

- Shielded transfers  
- Unlinkable deposit/withdraw  
- Works for native and mapped SPL assets  

---

## 🤝 Anonymous Referral System

Referral tracking without exposing:

- Inviter’s address  
- Invitee’s address  
- Transaction history  

Ideal for growth campaigns and loyalty programs.

## 🕸️ Anonymous Social Layer

A privacy-native on-chain social graph:
